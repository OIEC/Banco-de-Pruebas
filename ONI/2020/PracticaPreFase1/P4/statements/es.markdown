# Descripción

Halla el perímetro de un triángulo rectángulo que tiene lados $n$, $x$, $x+1$ en función de $n$ sabiendo que el lado $x+1$ es la hipotenusa del triángulo.

# Entrada

Un entero no negativo $n < 10000$.

# Salida

Un entero $k$ representando el perímetro del triángulo.

#Ejemplo

||input
3
||output
12
||description
Triángulo con lados 3, 4, 5.
||end

#Límites

$n < 10000$
