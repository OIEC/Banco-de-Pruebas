# Descripción

Crea un programa que recibe un entero no negativo $n$ e imprime la cantidad de parejas de enteros no negativos $(a, b)$ tales que $a^2+b^2 = n$.

# Entrada

Un entero no negativo $n < 10000$.

# Salida

Un entero $k$ representando la cantidad de parejas de enteros no negativos $(a, b)$ tales que $a^2+b^2 = n$.

#Ejemplo

||input
25
||output
4
||description
Pares: (0, 5), (3, 4), (4, 3), (5, 0).
||input
3
||output
0
||description
No hay pares (a,b) tales que a^2+b^2 = 3. 
||end

#Límites

$n < 10000$
